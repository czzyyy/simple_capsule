# simple_capsule
A very simple capsule in Tensorflow.Paper:[Dynamic Routing Between Capsules](https://arxiv.org/abs/1710.09829) 
# requirements
tensorflow 1.8.0  
numpy 1.15.2  
scipy 1.1.0  
matplotlib 3.0.0
